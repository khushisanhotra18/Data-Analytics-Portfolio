# Tata Motors Financial Performance Analysis

## Department
Finance

## Objective
To evaluate the financial performance of Tata Motors using income statement, balance sheet, and cash flow analysis over multiple years.

## Tools Used
- Python
- SQL
- Power BI

## Dataset
Web scrapped data from screener using python.

## Methodology
1. Structuring financial statements
2. Year-on-year comparison
3. Profitability and efficiency analysis
4. Cash flow interpretation

## Key Analysis Performed
- Revenue and net profit trend analysis
- Operating, investing, and financing cash flows
- Expense vs profitability evaluation
- Net profit margin analysis

## Key Insights
- Consistent increase in operating cash flows
- Negative cash flow from financing activities indicating debt repayment
- Net profit margin showed slight fluctuation year-on-year
- High expenses impacted net profitability despite strong sales

## Deliverables
- Python notebook
- Csv files of different table scrapped and cleaned
- SQL query for analysis
- Power BI dashboard
- PDF presentation explaining methodology and Key financial insights summary
