# Social Media App Performance Analysis

## Department
Marketing Analytics

## Objective
To analyze user engagement and content performance on a social media platform
to improve marketing and growth strategies.

## Tools Used
- Python
- Excel

## Dataset
Public dataset representing social media user and engagement activity.

## Methodology
Did the same procedure in both excel and python:
1. Data uploading and cleaning
2. Analyzing Data
3. Visualizing with different charts

## Key Analysis Performed
- User activity and engagement trends
- Post interaction metrics (likes, comments, shares)
- Platform usage patterns
- Content performance comparison

## Key Insights
- Engagement peaks during specific time slots on each platform
- Certain content formats generated higher interactions
- User retention varied significantly by age
- Marketing campaigns can be optimized using engagement data

## Deliverables
- Python script
- Excel analysis file and dashboard
- PDF presentation summarizing findings
