# Car Showroom Inventory Analysis

## Department
Operations

## Objective
To optimize inventory management by analyzing stock levels, sales status, and product movement across car models.

## Tools Used
- Excel
-  SQL
- Power BI

## Dataset
Database taken from kaggle i.e., publicly available.

## Methodology
1. Data cleaning and normalization
2. Inventory status classification
3. Stock vs sales performance analysis
4. Dashboard creation for inventory tracking

## Key Analysis Performed
- Stock availability by model and brand
- Fast-moving vs slow-moving vehicles
- Out-of-stock and overstock identification
- Inventory status color-coding (Red, Yellow, Green)

## Key Insights
- Certain models remained overstocked with low sales
- Fast-selling cars frequently faced low stock issues
- Inventory imbalance increased holding costs
- Real-time stock monitoring can reduce stockouts

## Deliverables
- SQL queries for inventory analysis
- 2 Power BI inventory dashboard 
- PDF presentation with recommendations and full project summary
